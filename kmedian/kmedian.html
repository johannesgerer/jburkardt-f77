KMEDIAN
A Program for the K-Medians Problem

KMEDIAN is a FORTRAN77 program which seeks solutions of the K-Medians problem, by G Cornuejols, M L Fisher, G L Nemhauser

Languages:

KMEDIAN is available in a FORTRAN77 version and a FORTRAN90 version.

Related Data and Programs:

KMEANS, a FORTRAN90 library which contains several different algorithms for the K-Means problem, which organizes a set of N points in M dimensions into K clusters;

Reference:

  1. G Cornuejols, M L Fisher, G L Nemhauser,
    Location of Bank Accounts to Optimize Float, an Analytic Study of Exact and Approximate Algorithms,
    Management Science,
    Volume 23, Number 8, pages 789-810, 1977.

Source Code:

  • kmedian.f, the source code.

Examples and Tests:

  • co04_dist.txt, a set of intercity distances.
  • kmedian.dat, an output file.
  • kmedian_output.txt, the output from a run of the sample program.

List of Routines:

  • MAIN is the main program for KMEDIAN.
  • SOL takes one step of the iteration.
  • GREEDY finds a nonoptimal solution using a greedy algorithm.
  • KWRITER gets input from the user.
  • SORT sorts the closest fund stock for each target stock.
